# Placeholder Practice Photos

These 8 images are shown when a prospect has fewer than 3 owner-uploaded photos.
They are replaced at launch with the practice's own photos.

## Required files

| Filename | Subject |
|---|---|
| `01-waiting-room.jpg` | Wachtkamer — licht, modern, uitnodigend |
| `02-treatment-room.jpg` | Behandelkamer — apparatuur zichtbaar maar niet beangstigend |
| `03-reception.jpg` | Receptiebalie |
| `04-exterior.jpg` | Gevel / entree van de praktijk |
| `05-equipment.jpg` | Detail van orthodontische apparatuur |
| `06-consultation.jpg` | Consult — orthodontist en patiënt in gesprek |
| `07-team-space.jpg` | Team-ruimte of gang |
| `08-detail.jpg` | Interieurdetail (plant, muur, logo, etc.) |

## Requirements

- Licensed for commercial use (Unsplash/Pexels both work — check per photo)
- Modern Dutch practice aesthetic: clean, light, not US-style clinics
- Consistent aspect ratio across all 8 (16:9 or 4:3 — pick one)
- Record the source URL + license below for each photo
